{% trans "1. Responsable du traitement" %}

{% blocktrans %} Parking.Belgium est une plateforme académique développée dans le cadre d'un travail de fin d'études. Elle permet la gestion centralisée des cartes de stationnement pour les 19 communes de la Région de Bruxelles-Capitale. {% endblocktrans %}

{% blocktrans %} Le responsable du traitement est l'auteur du projet, joignable par courrier électronique à l'adresse privacy@parking.belgium.local. Dans une mise en production réelle, ce rôle serait dévolu à la Région bruxelloise ou à chaque commune signataire. {% endblocktrans %}

{% trans "2. Base légale" %}

{% blocktrans %} Le traitement repose sur l'article 6.1.e du RGPD : « exécution d'une mission d'intérêt public ». La gestion des cartes de stationnement résidentielles est une compétence communale visant à organiser l'occupation rationnelle de l'espace public. {% endblocktrans %}

{% trans "3. Données collectées" %}

{% trans "Pour fonctionner, le service collecte :" %}

  • {% trans "Identité : nom, prénom, date de naissance, numéro de registre national." %}
  • {% trans "Coordonnées : email, téléphone, adresse postale (avec géocodage)." %}
  • {% trans "Véhicules : plaque d'immatriculation, marque, modèle, certificat d'immatriculation." %}
  • {% trans "Cartes : type (riverain, visiteur, professionnel), zones autorisées, validité." %}
  • {% trans "Paiements : montant, méthode, référence Stripe, statut. Aucun numéro de carte bancaire n'est stocké sur nos serveurs." %}
  • {% trans "Journaux techniques : adresse IP de connexion, dates, actions sensibles (audit RGPD)." %}

{% blocktrans %} Les plaques interrogées par l'API publique (scan-cars communaux) sont stockées sous forme de hash HMAC-SHA256 non réversible — la traçabilité est préservée sans exposer la plaque en clair dans les journaux. {% endblocktrans %}

{% trans "4. Destinataires" %}

  • {% trans "Agents et administrateurs habilités de votre commune." %}
  • {% trans "Sous-traitants techniques : Stripe (paiement), votre fournisseur SMTP (envoi d'emails). Aucune donnée n'est transmise à des tiers commerciaux." %}

{% trans "5. Durées de conservation" %}

  • {% trans "Compte actif : tant que l'utilisateur reste inscrit." %}
  • {% trans "Compte inactif : anonymisation après 3 ans sans connexion." %}
  • {% trans "Cartes de stationnement expirées : 10 ans (obligation comptable belge)." %}
  • {% trans "Paiements : 7 ans (obligation TVA)." %}
  • {% trans "Journaux d'audit : 3 ans (recommandation APD)." %}
  • {% trans "Codes visiteurs : 1 an après leur expiration." %}

{% trans "6. Vos droits" %}

{% trans "Conformément aux articles 15 à 22 du RGPD, vous disposez des droits suivants :" %}

  • {% trans "accès à vos données ;" %}
  • {% trans "rectification (modifiable directement depuis votre espace personnel) ;" %}
  • {% trans "effacement (« droit à l'oubli »), sous réserve des durées de conservation légales ;" %}
  • {% trans "limitation du traitement ;" %}
  • {% trans "portabilité (export machine de vos données) ;" %}
  • {% trans "introduction d'une réclamation auprès de l'Autorité de protection des données (APD) — autoriteprotectiondonnees.be." %}

{% blocktrans %} Pour exercer ces droits, écrivez à privacy@parking.belgium.local depuis l'adresse email associée à votre compte. Une réponse vous sera apportée dans un délai d'un mois. {% endblocktrans %}

{% trans "7. Sécurité" %}

{% blocktrans %} Les mots de passe sont hashés (algorithme PBKDF2). Les communications sont chiffrées en TLS. Les paiements transitent par Stripe (PCI-DSS niveau 1). L'accès aux données back-office est cloisonné par rôle (citoyen / agent / admin / super-admin) et toute action sensible est journalisée. {% endblocktrans %}

{% trans "8. Cookies" %}

{% blocktrans %} Le site utilise un unique cookie strictement nécessaire au fonctionnement (cookie de session pour vous garder connecté). Il ne fait l'objet d'aucun partage publicitaire ou analytique. Aucun cookie tiers n'est déposé. {% endblocktrans %}
